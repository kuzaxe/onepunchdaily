# Screenshots

Drop your three App Store screenshots here with these exact names and the
landing page will use them automatically (framed inside the phone mockups):

- `today.png`    — the Today screen
- `calendar.png` — the Calendar / streaks screen
- `charts.png`   — the Charts screen

Portrait PNGs straight from an iPhone are fine (roughly 9:19.5). Until these
files exist, the page falls back to built-in CSS re-creations of each screen,
so it never looks broken.
